This project built using HTML, CSS, and JavaScript, providing an interactive interface for selecting menu items. The app allows users to add items to their cart, view the total price, and fill in required personal data through a pop up modal, complete their order, and get a feedback message.

This page leverages JavaScript array methods such as map() and push() to dynamically manage menu items and orders. Using object destructuring, the code efficiently extracts values from objects. Interaction with HTML elements is handled through classList for adding and removing classes, and dataset is used to store custom data attributes. The page updates the display by modifying content dynamically based on user interaction. The join() method is utilized to format and display information, enhancing the user experience with efficient and clean coding practices.

Jimmy's Diner was created as part of Scrimba's Frontend Developer Career Path program, with the design provided through a Figma file.
