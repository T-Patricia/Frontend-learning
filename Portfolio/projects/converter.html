Unit Converterweb app

JavaScript

GitHub Repo Live version

The Unit Converter app allows users to convert between different units of length, volume, and mass. It utilizes JavaScript for DOM manipulation and performs dynamic conversions based on user input. The app converts meters to feet, liters to gallons, and kilograms to pounds.

Key JavaScript techniques, such as getElementById(), addEventListener(), and toFixed(), are used to handle user input and display results. The conversion outputs are formatted with template strings, ensuring an interactive and user-friendly experience.

Unit converter was created as part of Scrimba's Frontend Developer Career Path program, with the design provided through a Figma file.
